NUEVOS REQUERIMIENTOS - GENERADOR & AGENTE MODULAR
================================================================================

1. Agente Modular:
   - El sistema debe soportar múltiples roles (soporte, diseñador, ventas) mediante el parámetro clientId.
   - Cada rol tiene sus propias systemInstructions y herramientas (functions).

2. Rol "Site Designer":
   - Objetivo: Guiar al usuario para crear su web.
   - Tarea: Recolectar Nombre, Rubro, Colores, Descripción.
   - Salida: Generar archivos JSON de configuración y business data.
   - Tool: create_site_config(config_json, business_text).

3. Arquitectura de Generación:
   - Portfolio (Vercel): Frontend + Agente (Chat).
   - Worker (VPS/Docker): Ejecuta el proceso pesado (npm install, build, deploy).
   - Comunicación: Webhook/API entre Portfolio y Worker.

4. Preview en Tiempo Real:
   - El agente genera un JSON que se renderiza en un iframe antes del deploy final.
